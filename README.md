#  ShoeStock Manager
> **Sistema de Gestión de Inventario para Zapatería**

##  Descripción del Proyecto
Este proyecto es un MVP (Producto Mínimo Viable) desarrollado bajo la metodología Scrum para el **Parcial 3: Ciclo de Vida Ágil de Desarrollo**.

El sistema soluciona la problemática de pérdida de ventas por falta de información en bodega, permitiendo gestionar el stock de zapatos de manera digital, rápida y eficiente.

##  Funcionalidades Principales
El sistema cumple con las siguientes Historias de Usuario gestionadas en Jira:
*   **Registro de Productos:** Formulario para ingresar modelo, talla, precio y cantidad.
*   **Visualización de Stock:** Tabla dinámica que muestra el inventario actual.
*   **Persistencia de Datos:** Uso de `LocalStorage` (los datos no se borran al recargar la página).
*   **Gestión de Inventario:** Opción para eliminar pares vendidos o dados de baja.
